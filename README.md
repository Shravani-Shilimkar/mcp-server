# MCP Server Learning Journey 🚀

A hands-on project building Model Context Protocol (MCP) servers from scratch, phase by phase.

## Phases

| Phase | What I Built | Key Concepts |
|---|---|---|
| Phase 1 | Hello World server | Tools, Resources, Prompts |
| Phase 2 | Calculator + File Reader | Error handling, File system |
| Phase 3 | Notes + SQLite | Statefulness, Databases, SQL |
| Phase 4 | Weather + GitHub APIs | HTTP requests, Auth, Parallel calls |

## Tech Stack
- TypeScript
- MCP SDK (@modelcontextprotocol/sdk)
- Node.js
- sql.js (SQLite)
- open-meteo API (weather)
- GitHub REST API

## How to run any phase
```bash
cd mcp-phase1   # or phase2, phase3, phase4
npm install
npm run inspect  # opens MCP Inspector in browser
```
